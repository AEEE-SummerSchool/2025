# 2025年第八届新工科联盟暑期学校

## 📚 活动概述

为持续推进新工科人才培养工作，深化产学研协同创新，信息技术新工科产学研联盟可定制计算人才培养工作委员会定于2025年8月24日至25日，在西南交通大学峨眉校区举办第八届暑期学校。

本届暑期学校以 **"软硬协同·智能驱动"** 为主题，聚焦FPGA与AI PC两大领域，采用线上与线下相结合的混合模式，面向全国高校师生免费开放。
